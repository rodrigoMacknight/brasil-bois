# brasil-bois
A multi-plataform tradeable card game making fun of the sad political history of Brazil

# stack
Game:
 - Java
 - Libgdx
Server
 - node.js
 - aws
 - socket.io

# Developer guide
- Import into intellij
- Set android assets as resource source
- play desktop version
